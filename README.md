# ☸️ Kubernetes Bootcamp by TeraSkill

🚀 A structured, day-wise Kubernetes learning repository designed to take you from beginner to advanced with hands-on practice.

---

## 📌 What You'll Learn

- Kubernetes fundamentals
- Container orchestration concepts
- Kubernetes architecture (Control Plane & Nodes)
- Pods, ReplicaSets, Deployments
- Services & Networking
- ConfigMaps & Secrets
- Volumes & Storage
- Ingress & Traffic routing
- Scaling & Auto-healing
- Rolling updates & Rollbacks
- Real-world DevOps use cases

---

## 📂 Repository Structure

Each day contains:

- 📖 `notes.md` → Concept explanations  
- 🛠️ `tasks.md` → Hands-on exercises  

**Example structure:**
```
kubernetes-bootcamp/
│
├── Day01-Introduction/
│ ├── notes.md
│ ├── tasks.md
│
├── Day02-KIND-Pods/
│ ├── notes.md
│ ├── tasks.md
│
├── Day03-ReplicaSet-Deployment/
│ ├── notes.md
│ ├── tasks.md
│
└── README.md
```

## Who Is This For?

- DevOps beginners
- Students learning Kubernetes
- Docker users moving to Kubernetes
- Developers working with microservices
- Anyone preparing for DevOps interviews

## How to Use This Repo

1. Follow the days in order  
2. Read `notes.md`  
3. Complete `tasks.md`  
4. Practice on your local cluster (Minikube/KIND)  
5. Repeat & revise

## ⚙️ Prerequisites

- Basic Linux knowledge  
- Command line familiarity  
- Docker fundamentals (recommended)  
- System with Docker installed
